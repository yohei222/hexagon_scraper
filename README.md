# Hexagon Scraping

## 使用技術
* Ruby
* selenium-webdriver
* ChromeDriver

## 内容
僕の好きな音楽レーベルであるHexagon( https://www.hexagonhq.com/releases )から、曲名とURLをスクレイピングできます。
Excelで開くとこの写真のようになります。
